# Session Log

Append-only log of AI-assisted work sessions. Agents should read the last 10 entries at session start for context continuity.

## Format

Each entry follows this structure:

```
### YYYY-MM-DD — [Brief Task Title]
**Agent(s):** [which agents were used]
**What was done:** [1-2 sentence summary]
**What worked:** [techniques or approaches that succeeded]
**What didn't:** [approaches that failed, dead ends]
**Files changed:** [list key files modified]
**Open items:** [anything left unfinished]
```
